## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | USD to EUR Converter | Currency converter using real exchange rate logic |

| 03 | Contact Directory | Can add, search, edit delete and show contacts that were provided |
| 04 | Multiplier | simple program to multiply positive and/or negative numbers |
